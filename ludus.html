Concept

Ludus is an online fitness tracker. It aims at being concise and usefull without overwhelming the user with useless features.

Features

There are 3 main features. 1. A customizable profile that each athlete can build. 2. A daily fitness log where an athlete can log and track their work each day. 3. A goal setting and tracking system which allows each athlete to set and check off goals

Technologies

The frontend was built using HTML5, CSS, and Bootstrap. The backend was built using Node.js and couchDB.
